ПРАКТИЧЕСКАЯ РАБОТА №3 - ОТВЕТЫ НА ВОПРОСЫ:

1. Селектор по классу (.class) имеет специфичность (0,1,0) и может использоваться多次но, селектор по ID (#id) имеет специфичность (1,0,0) и должен быть уникальным на странице.

2. Каскад применяет стили в порядке: 1) Важность (!important), 2) Специфичность, 3) Порядок в CSS файле.

3. Расчет специфичности:
   - ul li.important = (0,1,2)
   - #site-title = (1,0,0)
   - .content > .featured-post = (0,2,0)

4. Наследуемые свойства: color, font-family, line-height, text-align. Ненаследуемые: background, border, margin, padding.

5. Селектор > выбирает только прямых потомков, а селектор через пробел - всех потомков любого уровня вложенности.

6. :hover для стилей при наведении, :nth-child() для выбора элементов по порядку. Полезны для интерактивных меню, таблиц, списков.

7. !important принудительно применяет стиль, игнорируя специфичность. Используется осторожно, так как усложняет переопределение стилей.

8. ::before и ::after добавляют декоративный контент до/после элемента. Могут содержать текст, иконки, декоративные элементы.

9. При одинаковой специфичности браузер применяет последнее правило в CSS файле.

10. Инструменты разработчика показывают примененные стили, специфичность, наследование, позволяют экспериментировать с изменениями в реальном времени.
