ПРАКТИЧЕСКАЯ РАБОТА №2 - ОТВЕТЫ НА ВОПРОСЫ:

1. Семантическая верстка использует теги с конкретным смыслом (header, article, nav), которые понятны браузерам и поисковым системам, тогда как несемантическая relies на универсальные div и span без смысловой нагрузки.

2. <article> используется для независимого, самодостаточного контента (рецепт пасты карбонара), а <section> - для тематической группировки внутри статьи (разделы "Ингредиенты", "Приготовление").

3. <header> и <footer> могут использоваться多次но на странице: для всего документа, а также для каждого <article> и <section>.

4. Для основного контента используется <main>, который указывает на главное содержимое страницы, в отличие от <div>, который не несет семантической нагрузки.

5. <nav> используется для основной навигации по сайту (главное меню), тогда как обычный список ссылок может использоваться для второстепенных ссылок. <nav> обязателен для основного меню навигации.

6. Атрибут datetime в теге <time> обеспечивает машинно-читаемый формат даты и времени, что важно для поисковых систем, календарей и accessibility.

7. <figure> и <figcaption> семантически связывают изображение с подписью, улучшая доступность для скринридеров и понимание контекста поисковыми системами.

8. <aside> используется для дополнительного, связанного контента (боковая панель с популярными рецептами, информацией о кухнях мира).

9. Семантическая разметка улучшает SEO через лучшее понимание структуры контента поисковыми системами и accessibility через корректную работу с скринридерами для людей с ограниченными возможностями.

10. Использованные семантические теги: header, nav, main, article, section, aside, footer, figure, figcaption, time, address. Они помогают пользователям и поисковым системам понимать назначение каждого элемента страницы.
